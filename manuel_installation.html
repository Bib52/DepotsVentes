Manuel d'installation

Prérequis :

Pour utiliser cette application vous devez avoir un ordinateur ou serveur avec :

  • apache et mysql installé;
  • composer pour installer les dépendances du serveur.

Installation

  • Télécharger le projet et dezipper le à la racine de votre serveur apache;
  • ouvrer un terminal et rendez vous dans votre serveur à l'emplacement "DepotsVentes/Server" puis faites la commande composer install;
  • Base de données :
    • connectez vous à votre serveur mysql puis créer la base de données "depotsventes";
    • importez le script "DepotsVentes/Server/depotsventes.sql" dans la base de données créé;
    • mettez vos informations de connexions à la base de données dans le fichier "DepotsVentes/Server/app/Configuration.php".
  • Tester sur "http://server_address/DepotsVentes/Client".
